## Getting Started

To get a local copy up and running follow these simple steps.

### Installation

1. Clone the repo

```sh
git clone https://github.com/MatthewDodi/sellit.git
```

2. Install NPM packages

```sh
npm install
```

3. Run the Development Server

```sh
npm run dev
```
